# YOLOv3
Real Time Object Detection using YOLOv3, Tensorflow, Keras and OpenCV

According to the paper [YOLOv3: An Incremental Improvement](https://pjreddie.com/media/files/papers/YOLOv3.pdf).

## Required Modules
- OpenCV
- Python    
- Tensorflow  
- Keras
- Numpy
- Matplotlib
